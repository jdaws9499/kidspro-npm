Allow/Block URL List (Optional)

The following list enables to override the web sites' self declared age ratings. You can add urls to block altdough they are allowed for the user's age. Or you can add urls to allow altdough they are not allowed for the user's age.

      Browse Schedule

      The browse schedule enables to add allwed time schedules to browse. You can add time and days of the week. It's allowed all the time as default. Uncheck box if you want to add certain allow times. If there is no schedule set, it's allowed all the time.

      Violated Access Logs

      You can monitor the history of visiting sites that are either beyond the user's age group, in the specified blocked URL list or outside the specified allowed browsing schedule.

      We can only keep minimal history for KidsPro as the browser storage is limited (5MB).
